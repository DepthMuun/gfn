Welcome to the GFN framework documentation. GFN (Geometric Flow Networks) is a physics-informed paradigm for sequential intelligence that reformulates sequence modeling through the lens of differential geometry and Hamiltonian mechanics.

## Key Features

- **O(1) Inference Memory**: Constant memory regardless of sequence length (G-SSM realization)
- **Symplectic Integration**: Energy-preserving numerical methods
- **Active Inference**: Uncertainty-aware dynamics adaptation
- **Functional Embeddings**: Scalable vocabulary representations
- **Geometric Stability**: Gradient stability over infinite horizons
